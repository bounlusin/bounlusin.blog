我的第一篇文章

这是我自己搭建blog写的第一篇文章。

从25号开始到现在，从什么都不会到成功，心里的满足感不言而喻。

这期间接触到的一些东西：

  • git

  • ruby

  • DevKit

  • jekyll

  • node.js

  • hexo

  • markdown

期间遇到不少困难，大部分还是吃了英文不好的亏。

原本走的是 git—>ruby—>devkit—>jekyll 这条路，但是jekyll安装过程中有个不明原因的错误，偶然发现了 hexo，于是转投 git—>node.js—>hexo ，效果不错。
