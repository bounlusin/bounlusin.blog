漫威电影顺序

1 《钢铁侠1》

《钢铁侠》是2008年上映的美国超级英雄电影。漫威电影宇宙的第一部影片，由乔恩·费儒执导，小罗伯特·唐尼及格温妮斯·帕特洛等人主演。

关键人物：钢铁侠

2《无敌浩克》

《无敌浩克》是2008年上映的超级英雄电影，漫威电影宇宙的第二部影片。根据漫威漫画改编，由路易斯·莱特里尔执。

关键人物：绿巨人

3《钢铁侠2》

《钢铁侠2》是2010年上映的超级英雄电影。漫威电影宇宙的第三部影片，由乔恩·费儒执导。

关键人物：钢铁侠

4《雷神1》

《雷神》是2011年上映的超级英雄电影，根据惊奇漫画改编，漫威电影宇宙的第四部影片。著名莎剧导演肯尼思·布拉纳执导，讲述了被放逐到地球的雷神托尔学习如何成为一个真正的英雄的故事。期间，雷神在地球上爱上了一个女科学家，在最后，雷神的好友来到地球寻找雷神回去拯救他们的国家，由于前行危险重重，雷神不得不与女友分离，但他答应会回来找女友。

关键人物：雷神托尔

5 《美国队长1》

美国漫画中最“主旋律”的超级英雄，非美国队长莫属。他用国名当做头衔，制服是红白蓝加上明亮的星星；一面同样颜色的盾牌就是他的武器。没错，他的诞生，正是源于爱国主义的目的。

1941年，第二次世界大战在欧亚战场打得如火如荼，远在大洋彼岸的美国此时却尚未参战。尽管如此，其国内反法西斯的呼声也越来越强烈。在这风雨欲来的多事之秋，及时漫画公司（Timely Comics，后来的Marvel）推出了一位爱国者，一位代表着美国精神的英雄——美国队长。

关键人物：美国队长

6 《复仇者联盟1》

《复仇者联盟》是2012年上映的超级英雄电影，也是漫威电影宇宙系列的第六部影片。由乔斯·韦登执导，小罗伯特·唐尼、克里斯·埃文斯、克里斯·海姆斯沃斯、马克·鲁法洛、斯嘉丽·约翰逊、杰瑞米·雷纳和汤姆·希德勒斯顿联袂出演。

片中讲述神盾局指挥官为了对付《雷神》中被流放的洛基，积极奔走寻找最强者，在神盾局斡旋下将钢铁侠、美国队长、雷神、绿巨人、黑寡妇和鹰眼侠六位超级英雄集结在一起，成立了“复仇者联盟”，共同携手应对邪神洛基。

关键人物：钢铁侠，绿巨人，美国队长，雷神，黑寡妇，鹰眼侠。

7 《钢铁侠3》

《钢铁侠3》是2013年上映的超级英雄电影。改编自漫威同名经典漫画，漫威电影宇宙的第七部影片，由沙恩·布莱克执导，小罗伯特·唐尼、格温妮丝·帕特洛、唐·钱德尔等参加演出。该片于2013年5月3日北美上映，同步登陆IMAX银幕。

《钢铁侠3》发生时间紧接在《复仇者联盟》结局的半年之后，讲述了托尼·斯塔克的生活被强敌毁灭殆尽，无路可退的他必须依靠精良的高科技装备以及过人才智，保护自己和身边最亲近的人，同时揪出真正的幕后元凶。

关键人物：钢铁侠

8 《神盾局》1~7集

神盾局（S.H.I.E.L.D.）是Marvel漫画中的组织，首次登场是在《奇异故事 #135》（1965年8月出版），由斯坦·李与杰克·科比创造。全称为国土战略防御攻击与后勤保障局（Strategic Homeland Intervention, Enforcement and Logistics Division），是美国政府专门用于同超能力英雄接洽并进行管理的特殊部队，现任指挥官为玛丽亚·希尔（Maria Hill），局长为尼克·弗瑞（Nick Fury）。它帮助钢铁侠组建了由众多超级英雄组成的复仇者联盟。

9 《雷神2》

《雷神2：黑暗世界》是由艾伦·泰勒执导，唐·佩恩、罗伯特·罗达特等编剧，克里斯·海姆斯沃斯、娜塔莉·波特曼、汤姆·希德勒斯顿等主演的电影，为2011年电影《雷神托尔》之续集，漫威电影宇宙的第八部电影。

影片描述纽约大战结束一年后，雷神索尔带着弟弟洛基回到仙宫面对远古黑暗势力的挑战。影片于2013年10月30日上映。

关键人物：雷神托尔

10 《神盾局》8~16集

11 《美国队长2》

《美国队长2》由漫威影业主席凯文·费吉制片，安东尼·罗素和乔·罗素兄弟执导，克里斯·埃文斯领衔主演。

故事发生在《复仇者联盟》纽约大战两年后，美国队长史蒂夫·罗杰斯已经定居华盛顿，努力适应现代社会的生活。

然而当一起神秘袭击事件伤害到神盾局的重要人物时，史蒂夫必须重新出马，与想要消灭他的刺客们抗争，和黑寡妇娜塔莎·罗曼诺夫一起找出幕后真凶，他们还必须招募有特训经验的桀骜英雄“猎鹰”加入他们的队伍。然而，在他们将面对的黑暗劲敌——寒冬战士的面罩背后，却是一个无比熟悉的面孔。
影片将于2014年4月4日在北美与中国同步上映。

关键人物：美国队长

12 《神盾局》17集

13 《银河护卫队》

《银河护卫队》（Guardians of the Galaxy）改编自漫威2008年的同名漫画，为预防银河系可能发生的灾难，星爵（Star-Lord）招募了一批星际英雄，将他们组建成银河护卫队。影片由詹姆斯·古恩执导，于2014年8月1日上映。

14 《复仇者联盟2》

复仇者联盟2即复仇者联盟：奥创时代。

在复仇者联盟奥创时代中，大反派为人工智能 - 奥创。而漫威也宣布奥创非漫画系列中的蚁人创造，奥创有着天才的智慧，他的外壳是由艾德曼合金制成，因此任何东西都无法伤害到它。他一次次被摧毁又一次次复活，产生过很多代，曾和钢铁侠、鹰眼、绯红女巫等都有过交集。猜测认为，《复联2》定名“奥创时代”，可能是奥创已经被定为大反派，因为他可以把所有的角色都贯穿起来，也有助于故事持续不断地往下发展。

在《复仇者联盟》中，神盾局（S.H.I.E.L.D.）的现任指挥官尼克·弗瑞招募了美国队长、钢铁侠、绿巨人、雷神索尔等超级英雄一起拯救世界。导演乔斯·韦登确认，《复仇者联盟2》中将加入快银和绯红女巫这两个角色。

MARVEL官方宣布，《复仇者联盟2》定名为：《The Avengers: Age of Ultron》（复仇者联盟：奥创纪元），片名logo也正式曝光。

关键人物：钢铁侠，绿巨人，雷神，黑寡妇，鹰眼侠。

15 《蚁人》

《蚁人》（Ant-Man）改编自Marvel的经典超级英雄漫画，由埃德加·赖特执导，预订于2015年全球公映，将作为漫威电影宇宙第三阶段的首发作品面世。“蚁人”汉克·皮姆由Marvel经典搭档斯坦·李和杰克·科比联手创造，于1962年初次在漫画中登场，作为一名出色的生化学家，他发现了一种可以将身体缩小至蚂蚁体积的微型原子，并制作一个特殊的头盔用以与昆虫交流。

生化学家汉克·皮姆（Hank Pym）打算用他最新发现的一组亚原子微粒，做一项可以改变大小的创造试验。结果中途出了些问题，拿自己做实验的汉克，竟然能够变得跟蚂蚁一样小。随后，他又发明了一项工具，帮助他跟蚂蚁们进行交流，甚至控制它们为自己所用。　在《钢铁侠》取得票房成功之前，英国导演埃德加·赖特曾为惊奇的漫画角色蚁人（Ant－Man）创作了一个电影剧本，但之后惊奇公司的兴趣又全部集中到“复仇者”系列上，《蚁人》就这样被搁置了起来。　不过上周迪士尼40亿美元买下惊奇公司以后，《蚁人》的命运也随之出现转机。在《娱乐周刊》的一篇报道中提及，迪士尼旗下的皮克斯（Pixar）已经在考虑制作一部《蚁人》的电影了。皮克斯方面并未做出正式表态，但外界猜测他们可能会继续使用埃德加·赖特的剧本。埃德加·赖特版的《蚁人》是一部带有喜剧色彩的科幻动作冒险片，而不是《亲爱的，我把孩子变小了》（Honey I Shrunk The Kids）那种家庭电影。

蚁人诞生于上世纪60年代，他是惊奇漫画中最迷你的超级英雄，变小后还可以通过特制头盔与昆虫交流。

关键人物：

斯科特·朗/Scott Lang（蚁人二代/Second Ant-Man）

亨利·皮姆/Henry Pym （蚁人一代/First Ant-Man）

据外媒2月20日报道，好莱坞漫威影业(Marvel Studios)开始为计划中的电影《奇异博士》寻觅导演。目前参与讨论的导演有执导《勇敢传说》的导演马克安德鲁斯、《血肉之躯》的导演乔纳森·莱文、《皇室风流史》的丹麦导演尼古拉哈·阿瑟以及即将上映的《改写人生》的新晋导演Dean Israelite。而《功夫熊猫》的编剧乔纳森·阿贝尔和格伦·伯杰也有望执笔该片编剧。

漫威影业此次花大手笔制作电影《奇异博士》，对该片寄予厚望。日前漫威的执行总裁对好莱坞记者说道，“《奇异博士》将会打破我们之前已经成文的标准，我们对本片形象孕育良久，他不同于我们之前做的任何形象。”

markdown学习总结

Markdown是什么

Markdown 是一种轻量级标记语言。它允许人们“使用易读易写的纯文本格式编写文档，然后转化成有效的XHTML(或HTML)文档”。

详情参见：wiki-Markdown

为什么使用Markdown

  • 纯文本，兼容性极强，可以用任何文本编辑器打开
  • 让使用者更专注于文字内容本身而不是排版
  • 格式转换方便，Markdown的文本可以轻松转换为html、电子书等
  • Markdown的标记语法与几号的可读性

Markdown语法

标题

在#  #之间填写标题内容

# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题

注：# 和「一级标题」之间建议保留一个字符的空格，这是最标准的 Markdown 写法。

列表

无序列表

- 文本1
- 文本2
- 文本3

有序列表

1. 文本1
2. 文本2
3. 文本3

注：-、1.和文本之间要保留一个字符的空格。

链接和图片

插入链接

[显示文本](链接地址)    
例如
[bounlusin的博客](http://bounlusin.github.io)

插入图片

![](图片链接地址)    
例如
![](http://7xivjb.com1.z0.glb.clouddn.com/luffy.jpg)

引用

在引用内容前面加上 > 即可
例如 > 发现更大的世界

注：> 和文本之间要保留一个字符的空格。

粗体和斜体

用两个*包含的文本即粗体
用一个*包含的文本即斜体

以上便是一些比较常用的Markdown语法。

我的第一篇文章

这是我自己用hexo搭建blog之后写的第一篇文章。

从25号开始到现在，从什么都不会到成功，心里的满足感不言而喻。

这期间接触到的一些东西：

  • git

  • ruby

  • DevKit

  • jekyll

  • node.js

  • hexo

  • markdown

期间遇到不少困难，大部分还是吃了英文不好的亏。

原本走的是 

git-->ruby-->devkit-->jekyll

这条路，但是jekyll安装过程中有个不明原因的错误，至今尚未查明。

偶然发现了 hexo，于是转投

git-->node.js-->hexo

效果不错。

2015/4/28 0:48:20

update：

其实昨晚并没有完全调试好，hexo生成的静态网页在我电脑上可以访问，却没有成功推送到github上，原因是

hexo deployer

这个指令总是失败。

查了资料发现hexo安装时忘了执行下面这个指令

npm install hexo-deployer-git --save

更正之后即可成功推送到github上，且不用

git push origin master

2015/4/28 17:31:53

不管怎么样，这个blog算是搭建起来了，以后我会在这里写一些学习总结，知识管理以及个人感悟。

为什么要写blog呢？

我觉得重要的不是一定要写出来什么东西给别人看，重要的是写blog过程中的自我思考和进步，当然写出来好的东西分享给大家也是好的。

有很多时候我们都不知所措，茫然，不知道该做什么，该怎么做，做的是否正确。自我思考对于我们来说，很重要。保持热情和毅力去做事固然重要，但停下来思考也是必不可少的。

写blog的目的就在于此：自我思考。
